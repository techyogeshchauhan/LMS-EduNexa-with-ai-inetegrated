üîê Security Features

Authentication Security

  • Flask-JWT-Extended Token System
  • Access Tokens (2 hours expiry)
  • Refresh Tokens (7 days expiry)
  • Token Blacklisting on Logout
  • Server Restart Invalidates Old Tokens
  • Multi-device Logout Support
  • Google OAuth Integration

Password Security

  • Werkzeug PBKDF2 SHA256 Hashing
  • Minimum 8 Characters Required
  • Uppercase & Lowercase Required
  • At Least 1 Digit Required
  • Special Character Required
  • Token-based Password Reset (1 hour)
  • Change Password with Current Password

API Security

  • Flask-CORS Configuration
  • JWT Required on Protected Routes
  • Role-Based Access Control (RBAC)
  • Input Validation & Sanitization
  • XSS & CSRF Prevention
  • MongoDB Parameterized Queries
  • Error Handling without Info Leakage

File Upload Security

  • File Type Validation (MP4, PDF, DOCX)
  • File Size Limits (100MB for videos)
  • UUID-based Unique Filenames
  • Secure Path Handling
  • Path Traversal Prevention
  • Enrollment Verification for Access
  • Separate Upload Directories
